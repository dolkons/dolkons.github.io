# Playwright: Начало работы

## Введение

Playwright - это современный инструмент для автоматизации тестирования веб-приложений. Он поддерживает все современные браузеры и предоставляет мощный API для написания надежных тестов.

## Установка

```bash
npm init playwright@latest
```

## Базовый пример

```typescript
import { test, expect } from '@playwright/test';

test('basic test', async ({ page }) => {
  await page.goto('https://example.com');
  await expect(page).toHaveTitle('Example Domain');
});
```

## Основные возможности

- Поддержка всех современных браузеров
- Автоматическое ожидание элементов
- Встроенная поддержка скриншотов
- Работа с iframe и вкладками
- Эмуляция мобильных устройств
