#### Автоматизация тестирования

Разработал framework для автоматизации тестирования на базе Playwright:

```javascript
import { test, expect } from '@playwright/test';

test.describe('Пример теста', () => {
  test('успешная авторизация', async ({ page }) => {
    await page.goto('/login');
    await page.fill('[data-testid="email"]', 'test@example.com');
    await page.fill('[data-testid="password"]', 'password123');
    await page.click('[data-testid="submit"]');

    await expect(page).toHaveURL('/dashboard');
  });
});
```

**Результаты:**
- Сокращение времени на регрессионное тестирование на 60%
- Покрытие критически важных пользовательских сценариев
- Интеграция с CI/CD пайплайном
